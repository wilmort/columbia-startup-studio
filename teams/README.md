# Team Submissions

Each team should create a folder with their team name and submit their work here.

## Folder Structure

```
teams/
├── team-awesome/
│   ├── README.md          # Team name + member UNIs
│   ├── ideas.md           # All ideas generated
│   ├── canvases.md        # Lean Canvas for top concepts
│   └── photos/            # Photos of sticky notes, FigJam, etc.
├── team-rocket/
│   └── ...
```
